## Installation

SMLinkPreview is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SMLinkPreview', :git => 'https://github.com/crspybits/SMLinkPreview.git'
```
